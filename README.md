# InversePINN-Airfoils
Descubrimiento de viscosidad turbulenta y parámetros efectivos en perfiles alares 2D mediante Inverse PINNs

## Datos del Proyecto

Debido al tamaño de los archivos, los datasets no están incluidos en el repositorio. 
Se pueden descargar los datos utilizados (crudos y procesador) en el siguiente enlace:

[Descargar Datasets (Google Drive)](https://drive.google.com/drive/folders/12NgoSwfnQ0zgHZEZwmGJLEFZro6wZ_--?usp=sharing)

### Instrucciones:
1. Descargar los archivos del enlace.
2. Agregar `ala_2d_completo.csv` dentro de la carpeta `datos/raw/`.
3. Agregar `ala_2d_processed_completo.csv` dentro de la carpeta `datos/processed/`.
